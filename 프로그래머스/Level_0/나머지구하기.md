# 문제 설명
## 정수 num1, num2가 매개변수로 주어질 때, num1를 num2로 나눈 나머지를 return 하도록 solution 함수를 완성해주세요.

<br>

# 제한사항
## 0 < num1 ≤ 100
## 0 < num2 ≤ 100
## 
## 

<br>

# 풀이

```js
function solution(num1, num2) {
    return (num1 % num2)
}
```

<br>

# 다른풀이

```js
const solution = (num1, num2) => num1 % num2
```
## 풀이 설명
### 화살표함수를 이용하여 1줄로 작성한게 인상깊다.
